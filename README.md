#placeholder.js

Let working "placeholder" in Internet Explorer (ie) 9 lte

support `<input type="text">`, `<input type="password">`, `<textarea>` tag

not a jQuery Plugin


## How to use

import placeholder.js like this
```html
<script type="text/javascript" src="placeholder.js"></script>
```

that's it
